Learning React JS with WPU

1. What is react ? 
	- react is library javascript for creating a (antarmuka) user (UI) yang interaktif
	- Antarmuka user is a element pada layer yang dapat show and interaction with user
 	- Disebut library because menyediakan function for membangun UI, but whatever developer for used this function
	- Un-opinionated (cara menggunakannnya tidak teralu diatur)
	- menyebabkan ekosistem berkembang	
	- Tools pihak kegitanya banyak

2. Latihan create code Javascript 
	- Membuat code sederhana dengan html dan js untuk menampilkan beberapa tampilan

3. React js 
	- Mempersingkat penulisan js dalam menampilkan beberpaa komponen didalam html dan javascript
	- Membuat code html dengan react

4. Prerequisite (yang harus disiapkan untuk learning react)
